# 简单GTO

自用德州扑克 8 人现金局实战训练器。V1 固定规则：8-max，盲注 10/20，200BB 入场，手机竖屏优先，策略表只服务翻前。

## 当前能力

- 8 人现金桌，固定玩家座位，BTN/SB/BB/UTG 等位置按手轮换。
- 完整手牌流程：翻前、翻牌、转牌、河牌、摊牌、弃牌提前结束、全下自动发牌。
- 机器人桌风：顶级强手 + 偏离型真人风格，局后展示个性和数据。
- 翻前范围表：底部 `策略` 或座位旁扑克牌图标打开，只在翻前可用。
- 现金局补码：只在手与手之间补满到 200BB。
- 抽水：5%，封顶 3BB；No flop, no drop；未跟注筹码不参与抽水。
- 本场总结：机器人数据、本场结果、最近手牌、最大问题。

## 本机验证

```powershell
bun run verify
```

验证内容包括：

- `bun test` 规则测试。
- 浏览器入口打包检查。

如果 `.tmp-bun-build` 被留下，可以直接删除；它已加入 `.gitignore`。

## 本机预览

```powershell
bun run serve
```

脚本会输出：

- 本机地址：`http://localhost:4173`
- 同 Wi-Fi 手机可尝试访问的局域网地址。

如果 Windows 当前网络栈报“无法加载或初始化请求的服务提供程序”，这是本机网络服务问题，不代表 App 代码坏了。

## 手机使用路径

最简单试打路径：先在电脑运行 `bun run serve`，手机和电脑连同一个 Wi-Fi，用脚本输出的局域网地址打开。

如果要在安卓上像 App 一样添加到桌面，通常需要 HTTPS 静态托管；本项目是纯静态 Web App，后续可直接放到 Vercel、Cloudflare Pages、Netlify 或 GitHub Pages。
