# 简单GTO · V2.6 全强牌手桌池 PRD

> 生成日期：2026-05-13
> 状态：已确权，进入实现

## 1. 背景

用户明确最新目标：简单GTO 的同桌 7 个 AI 都要有很强打牌能力，风格可以不同，但不应该再出现鱼、紧弱、疯鱼、普通娱乐玩家这类弱玩家定位。

此前 V2/V2.3 为了模拟常见现金桌，保留了部分漏洞型玩家。这个方向已经不符合当前训练目标。V2.6 将桌池从“真实混合桌”切换为“全强训练桌”。

## 2. 目标

- 7 个 AI 都是强牌手。
- 可见标签不再出现弱玩家或鱼向描述。
- 保留风格差异：TAG、平衡、压力、紧防、LAG、摊牌控制、高波动、现场直觉。
- 旧 key 保留，用于兼容本地存档和已有测试入口。
- 策略参数同步加强，不能只改名字。

## 3. 牌手风格

- 强 Regular · TAG：范围干净，价值下注稳定。
- 强 Regular · 平衡型：基准训练对手。
- 强 Regular · 攻击型：主动权和多街施压更强。
- 强 Regular · 紧防型：入池更窄，但不再过度软弱。
- 强 Regular · LAG 压迫型：宽范围、高压迫，但仍有牌面约束。
- 强 Regular · 摊牌控制型：防守和 bluff catch 能力强，不是跟注站。
- 强 Regular · 高波动攻击型：大尺度和高频施压，但不是无脑疯鱼。
- 强 Regular · 现场直觉型：混合 live-pro 风格，不再是普通娱乐玩家。

## 4. 实现范围

- 更新桌池可见标签。
- 更新行为参数、翻前 profile、翻后 profile。
- 桌池强度从 soft / standard / tough 改为 strong / tough / elite。
- 增加测试，确保当前可见标签不含弱玩家词汇。

## 5. 非目标

- 不改存档 key。
- 不引入 solver。
- 不做云端模型。
- 不改桌面快捷按钮。

## 6. 验收标准

- `bun run verify:ui` 通过。
- 当前桌池 7 个 AI 标签全部包含“强”。
- 当前桌池标签不包含“弱”“鱼”“娱乐”“跟注站”等弱玩家词汇。
- LAG / 紧防 / 摊牌控制 / 高波动仍表现出不同打法。
- 桌面快捷按钮仍打开最新主项目目录。
