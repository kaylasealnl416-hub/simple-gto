简单GTO · V2.5 手牌复盘摘要 PRD

生成日期：2026-05-13 · 状态：已确权，进入实现

背景

V2.3 已提升 AI 牌手差异，V2.4 已固化 UI 冒烟验收。当前用户打完一手后能看到结果和历史，但训练价值还偏弱：用户需要知道“这手应该记住什么”，而不是只看谁赢了多少钱。

目标

  • 每手结束时生成一句复盘摘要。
  • 复盘进入“选项 / 手牌历史”和“本场总结 / 最近完成的手牌”。
  • 如果本手已有明显错误，优先展示错误与下一步建议。
  • 如果无明显错误，按结果给出简短训练反馈。
  • 不引入 solver，不做长篇讲解，不打断当前牌桌节奏。

范围

  • 新增纯函数 buildHandReview 生成复盘数据。
  • 早期弃牌结束和摊牌结算都写入复盘。
  • 历史列表展示复盘 headline 与 next step。
  • 单元测试覆盖错误手、非摊牌赢池、摊牌输池。

验收标准

  • bun run verify:ui 通过。
  • 手牌历史中最近 6 手能展示复盘摘要。
  • 本场总结最近完成手牌能展示同样复盘摘要。
  • 有 mistake 的手牌优先显示“偏离建议”。
  • 无 mistake 时仍显示简短的结果型训练建议。
