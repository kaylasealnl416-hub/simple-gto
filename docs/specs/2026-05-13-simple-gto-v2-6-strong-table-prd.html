简单GTO · V2.6 全强牌手桌池 PRD

生成日期：2026-05-13 · 状态：已确权，进入实现

背景

用户明确最新目标：简单GTO 的同桌 7 个 AI 都要有很强打牌能力，风格可以不同，但不应该再出现鱼、紧弱、疯鱼、普通娱乐玩家这类弱玩家定位。

此前 V2/V2.3 为了模拟常见现金桌，保留了部分漏洞型玩家。这个方向已经不符合当前训练目标。V2.6 将桌池从“真实混合桌”切换为“全强训练桌”。

目标

  • 7 个 AI 都是强牌手。
  • 可见标签不再出现弱玩家或鱼向描述。
  • 保留 TAG、平衡、压力、紧防、LAG、摊牌控制、高波动、现场直觉等风格差异。
  • 旧 key 保留，用于兼容本地存档和已有测试入口。
  • 策略参数同步加强，不能只改名字。

验收标准

  • bun run verify:ui 通过。
  • 当前桌池 7 个 AI 标签全部包含“强”。
  • 当前桌池标签不包含“弱”“鱼”“娱乐”“跟注站”等弱玩家词汇。
  • LAG / 紧防 / 摊牌控制 / 高波动仍表现出不同打法。
  • 桌面快捷按钮仍打开最新主项目目录。
